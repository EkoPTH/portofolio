Profil singkat saya

Saya adalah Lulusan dari Institut Sains dan Teknologi Nasional jurusan Teknik Informatika, dan saya aktif dalam berorganisasi baik di Media sosial Telegram dan di Dunia Nyata. Sebagai mantan aktivis anggota Resimen Mahasiswa berlatar belakang IT saya berjanji pada diri saya bahwa saya harus dapat mengimplementasikan dan aktif turut membantu lingkungan bedasarkan disiplin ilmu dan hal-hal yang sudah saya dapat dari tempat saya belajar dan hal-hal baru didalam komunitas saya. Sebagai penutup saat ini saya tergabung dalam komunitas telegram frontend developer Indonesia sebagai anggota, Indonesia Honeynet Project sebagai aktivis Malware Analyst, dan Indonesia Level 7 Team sebagai mantan kepala bagian Penetration Test

FrontEnd Developer

Saya sangat senang beraktivitas koding dalam membangun sebuah Website baik statis maupun dinamis dan secara mandiri saya melakukan tes penetrasi dan memperbaiki setiap Bug yang ditemukan pada web buatan saya. Berbekal wawasan pengetahuan yang saya dapatkan di Komunitas saya, cukup banyak project website yang sudah saya buat.

Wordpress Developer

Malware Analyst
